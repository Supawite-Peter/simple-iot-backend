# Simple IoT Back-end

## Description

A simple IoT back-end server based on nest.js framework.

## Features

- Users Authentication API
- Users Management API
- Devices Management API
- Devices Status API

## To-Do

- SQL/noSQL database integration
- API Swagger Page
- Devices Status MQTT Support
- GraphQL integration

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
